# backend.md

This file is an overview of what the backend needs to do. 

## Team linking

- Each person should get a unique code for to sync. teams will choose the main phone, and then use that phones unique code to sync all information 
    - information includes:
        - match timings
            - includes teams in each match as well as times
        - different teams in the event
        - teams on watchlist
        - any scout reports
        - etc.
- anyone that have their websites linked should be able to make edits to the schedule, and the other things and it will update on other peoples phones. 

## Home Page

- On the home page, the backend needs to sync:
    - the reminders
- the home page should pull information from other tabs, such as match schedule and information, as well as the 'scout next' section. 

## Schedule Page

- as expected, the matches should sync between different phones. 
    - should sync teams in each match
    - should sync the match number, and match time

## Watch List
- should sync what teams that are on the watch list
